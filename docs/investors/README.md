# TreasuryOS Investor Materials

This folder contains reusable investor-facing material for introducing TreasuryOS
without overstating the current product posture.

## Recommended usage order

1. Start with the executive summary.
2. Use the messaging guide for meetings, email outreach, and short answers.
3. Build or refresh the deck from the slide outline.
4. Use the future-work document to frame the roadmap clearly.
5. Use the FAQ for diligence conversations.
6. Use the video script for a recorded founder intro or product overview.

## Current product guardrails

- Say **pilot-live**, not broad production rollout.
- Say **testnet-active Solana sync**, not mainnet live.
- Say **AI advisory-only**, not autonomous approvals or decisioning.
- Say **external audit prepared**, not externally audited yet.
- Say **production KYC and banking integrations are still ahead**, not already live.
