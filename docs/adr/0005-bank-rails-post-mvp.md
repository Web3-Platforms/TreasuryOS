# ADR 0005: Bank Rails Post-MVP

## Status

Accepted

## Decision

AMINA and SWIFT work are explicitly post-MVP.

## Rationale

- live bank rails would dominate the schedule
- pilot value comes first from compliance workflow control and evidence
- the current product does not yet need production settlement integrations to validate demand

## Consequence

`apps/bank-adapter` stays in the repo as future work but should not block or dilute MVP delivery.
